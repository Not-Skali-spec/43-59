Pętla FOR

  • Konstrukcja: for (i = 0; i < n; i++)
  • Opis: Pętla z określoną liczbą powtórzeń
  • Schemat blokowy:
    1. Start
    2. Inicjalizacja
    3. Warunek
    4. Instrukcja
    5. Modyfikacja
    6. Koniec

Pętla WHILE (kontrolowane wejście)

  • Konstrukcja: while (warunek)
  • Opis: sprawdza warunek przed wejściem do pętli
  • Schemat blokowy:
    1. Start
    2. Sprawdzenie warunku
    3. Jeśli TAK → wykonaj
    4. Jeśli NIE → koniec

DO WHILE (kontrolowane wyjście)

  • Konstrukcja: do { } while (warunek)
  • Opis: wykonuje się przynajmniej raz
  • Schemat blokowy:
    1. Start
    2. Wykonaj instrukcję
    3. Sprawdź warunek
    4. Jeśli TAK → powtórz
    5. Jeśli NIE → koniec
